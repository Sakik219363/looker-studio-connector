# Looker Studio Connector for Matomo

## Useful commands

Installing clasp:

```
$ npm install @google/clasp -g
$ clasp login
```

### Creating the App Script instance

```
$ npm run create
```

### Push to apps script

```
$ clasp push
```

### Deploy the currently pushed code to Apps Script

```
$ clasp deploy # this will automatically create a new version
```
